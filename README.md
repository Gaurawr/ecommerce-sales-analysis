# 🛒 Ecommerce Sales Analysis

## Overview
Analysis of real ecommerce sales data combining 
customer details and basket information to derive 
actionable business insights.

## Tools Used
- 🗄️ SQL — Data querying and analysis
- 📊 Power BI — Interactive dashboard
- 📁 CSV — basket_details + customer_details

## Dataset
- Total Orders: 15,000+
- Unique Customers: 13,870+
- Total Items Sold: 32,000+

## Key Insights
- Average 2 items per order
- Gender wise purchase behaviour analysed
- Top 10 products identified by units sold
- Customer age and tenure impact on sales
- Monthly sales trend analysis

## Dashboard Features
- KPI Cards — Orders, Customers, Items Sold
- Top 10 Products by Sales
- Sales by Gender
- Monthly Sales Trend
- Customer Age Distribution
- Customer Tenure Analysis

## Skills Demonstrated
- SQL JOINs across multiple tables
- Power BI relationships and data modelling
- DAX measures
- Business insight generation
